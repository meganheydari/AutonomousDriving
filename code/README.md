# Code Readme

Included files:

2 ESP's used. One, `Q6QuestHub/` contained IR reciever, 12c_display for split times, and a udp client

The other ESP, `Q6QuestCrawler/` controlled all functionality for driving and steering the Crawler

`jsWebserver/` was a node js webserver used as an intermediary for both esp's and the camera, it is hosted on the pi.
